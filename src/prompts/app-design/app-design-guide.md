# 习惯养成 App 原型设计

### 项目概述
基于《掌控习惯》核心原理的安卓 App 原型设计，帮助用户通过科学方法培养好习惯。

### 核心功能规划

#### 1. 主要功能模块
- **习惯追踪和记录**: 每日习惯完成情况记录
- **四大定律实践**: 明显、吸引、简单、满意
- **习惯统计和分析**: 可视化数据展示
- **目标设定与进度跟踪**: 长期目标管理
- **社区激励系统**: 用户间的互动和鼓励
- **成就系统**: 徽章和奖励机制

#### 2. 主要界面规划

| 界面名称 | 功能描述 | 优先级 |
|---------|---------|--------|
| 引导页/登录页 | App 入口，用户身份验证 | P0 |
| 首页（习惯概览） | 展示用户的所有习惯和今日完成情况 | P0 |
| 习惯详情页 | 查看单个习惯的详细信息和历史记录 | P0 |
| 添加/编辑习惯页 | 创建新习惯或修改已有习惯 | P0 |
| 数据统计页 | 习惯完成率、连续天数等统计分析 | P1 |
| 社区页面 | 浏览和分享其他用户的习惯故事 | P1 |
| 个人中心 | 个人资料、设置和成就查看 | P1 |
| 设置页面 | 应用设置、通知、数据备份等 | P2 |

### 界面设计要素

#### 设计风格
- **风格定位**: 极简科技感 + 亲和力
- **配色基调**: 
  - 主色调：清爽蓝（#2E6FFF）象征稳定和专注
  - 辅助色：活力橙（#FF6B6B）表示激励和完成
  - 中性色：深灰（#2C3E50）和浅灰（#ECF0F1）
  
#### 交互亮点
- **无干扰设计**: 最小化通知弹窗
- **流畅动画**: 页面切换和操作反馈
- **手势操作**: 滑动编辑、长按菜单等
- **视觉反馈**: 清晰的状态变化提示

### 核心功能详解

#### 1. 引导页面设计
```
- 品牌 Logo 和应用标题
- 简洁的价值主张文本
- "开始使用" 按钮
- 可选的注册/登录选项
```

#### 2. 首页（习惯概览）
```
布局结构：
- 顶部：日期和当日进度环
- 中部：习惯卡片列表
  - 习惯名称和描述
  - 完成情况指示（打卡按钮）
  - 连续天数徽章
  - 完成率百分比
- 底部：浮动按钮（添加习惯）

卡片设计：
- 未完成状态：浅灰色背景，打卡按钮
- 已完成状态：绿色勾选标记，完成时间显示
- 悬停效果：卡片轻微放大，阴影增强
```

#### 3. 添加习惯页面
```
表单字段：
1. 习惯名称（必填）
2. 习惯描述（可选）
3. 习惯类别选择
4. 四大定律配置
   - 明显：设置提醒时间
   - 吸引：选择激励方式
   - 简单：设置目标频率（日/周/月）
   - 满意：选择奖励方式
5. 目标设置
   - 目标频率（每天/每周等）
   - 持续天数

交互特性：
- 实时表单验证
- 保存草稿功能
- 清晰的帮助文本
```

#### 4. 数据统计页面
```
展示内容：
1. 完成率统计
   - 今日完成率（圆形进度条）
   - 本周完成率（条形图）
   - 本月完成率（曲线图）

2. 长条图表
   - 最近 30 天完成情况日历热力图
   - 连续打卡最长记录

3. 成就展示
   - 已解锁徽章
   - 下一个目标里程碑

4. 对比分析
   - 与上周/上月的对比
   - 习惯间的完成率对比
```

#### 5. 社区页面
```
功能模块：
1. 发现页
   - 他人的成功故事
   - 热门习惯推荐
   - 励志分享

2. 我的分享
   - 分享个人成就
   - 邀请朋友加入
   - 分享设置（公开/仅朋友/私密）

3. 互动功能
   - 点赞和评论
   - 关注用户
   - 排行榜展示
```

#### 6. 个人中心
```
内容分布：
1. 用户信息区
   - 头像和用户名
   - 签名和简介
   
2. 成就展示区
   - 解锁的徽章墙
   - 总习惯数
   - 连续打卡记录
   
3. 快捷操作区
   - 数据导出
   - 社交分享
   - 设置入口

4. 账户管理
   - 编辑个人信息
   - 修改密码
   - 数据备份
   - 退出登录
```

### 设计规范

#### 色彩体系
```
主色调：#2E6FFF (清爽蓝)
辅助色：#FF6B6B (活力橙)
成功色：#51CF66 (成功绿)
警告色：#FFD43B (警告黄)
危险色：#FA5252 (危险红)

背景色：#FFFFFF 或 #F8F9FA
文本色：#2C3E50 (深灰) / #7F8C8D (浅灰)
```

#### 字体规范
```
标题：20px, 700 粗细
副标题：16px, 600 粗细
正文：14px, 400 粗细
辅助文本：12px, 400 粗细
```

#### 间距规范
```
基础单位：8px
常用间距：8px, 12px, 16px, 20px, 24px, 32px
```

#### 圆角规范
```
小：4px (输入框、标签)
中：8px (卡片、按钮)
大：16px (模态框、底部弹窗)
```

### 用户流程

#### 新用户入门流程
```
1. 安装 App
2. 查看引导页
3. 用户注册/登录
4. 个性化设置
5. 添加第一个习惯
6. 开始每日打卡
7. 查看数据分析
```

#### 日常使用流程
```
1. 打开 App
2. 查看首页习惯列表
3. 完成今日习惯打卡
4. 查看完成进度
5. 浏览社区励志故事
6. 查看个人成就和数据
```

### 开发建议

#### 技术栈
- **平台**: Android Studio (Kotlin/Java)
- **UI框架**: Material Design 3
- **数据库**: SQLite / Room
- **网络**: Retrofit / OkHttp
- **状态管理**: ViewModel + LiveData

#### 数据库设计
```sql
-- 用户表
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    username TEXT,
    email TEXT,
    created_at TIMESTAMP
);

-- 习惯表
CREATE TABLE habits (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    name TEXT,
    description TEXT,
    category TEXT,
    created_at TIMESTAMP
);

-- 打卡记录表
CREATE TABLE check_ins (
    id INTEGER PRIMARY KEY,
    habit_id INTEGER,
    date DATE,
    completed BOOLEAN,
    notes TEXT,
    created_at TIMESTAMP
);
```

### 成功指标

- **活跃度**: DAU / MAU
- **留存率**: 7日、14日、30日留存率
- **完成率**: 用户习惯完成率平均值
- **社交互动**: 分享和点赞数

## 工具导航宝箱项目

### 项目概述
为用户提供一个便捷的工具导航和管理平台，支持小程序和 H5 平台部署。

### 核心需求

#### 1. 用户需求分析
- 快速发现和访问各类实用工具
- 个性化工具收藏和分类
- 工具推荐和评分
- 跨平台无缝使用

#### 2. 功能模块规划

| 模块名称 | 核心功能 | 优先级 |
|---------|---------|--------|
| 首页导航 | 工具分类展示和搜索 | P0 |
| 分类浏览 | 按类别浏览工具列表 | P0 |
| 工具详情 | 工具介绍、评分和使用 | P0 |
| 收藏夹 | 保存常用工具 | P0 |
| 工具搜索 | 全文搜索和筛选 | P1 |
| 用户评分 | 工具评价和评论 | P1 |
| 推荐系统 | 个性化工具推荐 | P2 |
| 我的页面 | 个人收藏和历史记录 | P1 |

#### 3. 界面原型设计

##### 首页布局
```
┌─────────────────────────────────────┐
│      搜索栏 + 筛选按钮              │
├─────────────────────────────────────┤
│ 分类标签 (横向滑动)                 │
├─────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌───────┐│
│ │ 工具卡片 │ │ 工具卡片 │ │  ...  ││
│ │ 名称     │ │ 名称     │ │       ││
│ │ 描述     │ │ 描述     │ │       ││
│ │ ⭐ 评分  │ │ ⭐ 评分  │ │       ││
│ └──────────┘ └──────────┘ └───────┘│
├─────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌───────┐│
│ │ 工具卡片 │ │ 工具卡片 │ │  ...  ││
│ └──────────┘ └──────────┘ └───────┘│
├─────────────────────────────────────┤
│ 底部导航 (首页 / 搜索 / 收藏 / 我的)│
└─────────────────────────────────────┘
```

##### 搜索和筛选
```
- 搜索框：支持工具名称和关键词搜索
- 筛选选项：按分类、评分、更新时间排序
- 搜索历史：显示最近搜索的工具
- 热搜榜单：展示最热门的工具
```

##### 工具详情页
```
┌──────────────────────────────────────┐
│ 工具名称 + 评分 + 收藏按钮           │
├──────────────────────────────────────┤
│ 工具图标和截图                      │
├──────────────────────────────────────┤
│ 工具描述和功能介绍                   │
├──────────────────────────────────────┤
│ 立即使用 按钮 (跳转工具链接)         │
├──────────────────────────────────────┤
│ 评分和评论                           │
│ ┌──────────────────────────────────┐│
│ │ 用户评价和留言                    ││
│ │ 日期和评分                        ││
│ └──────────────────────────────────┘│
├──────────────────────────────────────┤
│ 我也想评价 输入框                    │
└──────────────────────────────────────┘
```

### 设计规范

#### 色彩系统
```
主色调：#6366F1 (靛蓝色) - 专业、可信
辅助色：#F97316 (橙色) - 温暖、友好
成功色：#22C55E (绿色) - 确认、完成
```

#### 卡片设计
```
- 白色背景，圆角 8px
- 工具图标：圆形，48px × 48px
- 工具名称：16px, 600 粗细
- 工具描述：14px, 400 粗细，最多 2 行
- 评分展示：⭐ 4.5 (共 233 评价)
- 悬停效果：轻微放大、阴影增强
```

#### 响应式设计
```
移动端 (320px-480px)：单列卡片
平板端 (481px-768px)：两列卡片
桌面端 (769px+)：三到四列卡片
```

### 开发建议

#### 技术选择
- **小程序**: 微信小程序框架 (Taro 或 uniapp)
- **H5**: React + TypeScript + Tailwind CSS
- **后端**: Node.js/Express 或 Python/FastAPI
- **数据库**: MongoDB 或 PostgreSQL

#### 功能实现优先级
1. 工具展示和搜索 (P0)
2. 收藏功能 (P0)
3. 用户评分评论 (P1)
4. 推荐系统 (P2)
5. 统计分析 (P2)
