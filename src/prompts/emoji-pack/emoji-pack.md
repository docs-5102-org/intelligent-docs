# 🐶 示例一：整套宠物表情包（可爱柴犬主题）

```
为微信表情包创作一套 24 张宠物表情图，
主角为一只圆脸柴犬，Q版卡通风格，线条干净，配色温暖橙色系，
每张尺寸 240×240 px，PNG 格式，背景透明，
角色始终居中，构图简洁，不超出边界，
每张表达一个清晰情绪，可独立使用，
避免复杂背景，不包含敏感或侵权内容。

情绪包括：
开心、委屈、生气、无语、加油、谢谢、晚安、困了、害羞、震惊、
点赞、鼓掌、爱心眼、流泪、思考、摸鱼、干饭、催促、收到、OK、
拜托、红包拿来、加班崩溃、摸摸头。

风格统一，角色比例一致。
```

```
PNG 透明背景，
画面简洁，适合微信聊天使用。
```

# 🐰 示例三：无文字纯表情版本（更安全上架）

```

夸张大眼睛，柔和粉色系，
无文字，仅通过动作表达“害羞”情绪，
240×240 px PNG，
透明背景，
构图居中，不裁切角色边缘。
```
