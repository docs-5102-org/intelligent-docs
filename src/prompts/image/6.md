# ⚙️ 高级技巧与通用词库-6

本文档收录 AI 图片生成的高级技巧、通用质量词、风格参考词及负向提示词，适合进阶使用者参考。

---

## 一、提示词黄金法则

> **好的提示词 = 清晰的视觉想象 + 专业的术语 + 足够的细节 + 合适的风格**

### 1. 清晰具体

❌ 不好：`美女`

✅ 好：`一位 25 岁的优雅东方女性，长黑色柔顺长发，穿着浅粉色旗袍`

### 2. 丰富细节

❌ 不好：`在房间里`

✅ 好：`在现代精致的卧室中，有大落地窗，窗外是城市夜景，室内有温暖的琥珀色灯光照射`

### 3. 优先级排序

把最重要的描述放在最前面，格式建议：

```
[最重要内容], [重要细节], [环境], [光线], [质量修饰符], [风格]
```

## 七、负向提示词（Negative Prompts）

告诉模型你不希望出现的元素：

```
ugly, distorted, deformed, blurry, low quality,
watermark, text, artifacts, mistakes, errors,
bad anatomy, wrong hands, extra fingers,
poorly drawn, underdeveloped, out of frame
```

---

## 八、权重调整（部分模型支持）

使用括号调整关键词的权重：

```
(beautiful eyes:1.5)   # 增加对漂亮眼睛的强调（权重1.5倍）
(detailed hair:0.8)    # 减少对头发细节的强调（权重0.8倍）
```

## 九、迭代优化示例

**第一版（基础）：**

```
A beautiful girl with long hair in a garden
```

**第二版（添加细节）：**

```
A beautiful 25-year-old girl with long flowing black hair, bright expressive eyes,
wearing a pink dress, standing in a blooming garden with roses
```

**第三版（完整优化）：**

```
(masterpiece, best quality, ultra-detailed),
a beautiful 25-year-old girl with long flowing black hair,
bright expressive eyes, wearing an elegant pink dress,
standing gracefully in a blooming garden filled with red roses,
soft golden sunlight filtering through, dreamy atmospheric lighting,
professional digital painting, trending on artstation
```

## 十、快速构建流程

1. **确定主体** → 人物 / 物品 / 场景
2. **添加特征** → 外观 / 气质 / 风格
3. **设置环境** → 背景 / 场景 / 道具
4. **指定风格** → 艺术风格 / 质感
5. **加入光线** → 照明方式 / 氛围
6. **质量增强** → 质量修饰符 + 艺术参考
7. **测试和迭代** → 调整和优化
